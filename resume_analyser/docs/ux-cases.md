## Кейсы

### Добавить новые резюме \ вакансии

Слева в меню нажимаю "добавить вакансию" или "добавить резюме". 
        Без ограничения общности далее рассмотрим на примере вакансий.

Вставляю ссылку на вакансию и нажимаю "добавить".

После нажатия "добавить" извлекается текст со страницы и отправляется в модель для извлечения данных, а именно:
    - название вакансии
    - краткого описания вакансии

Когда модель отдала ответ, он сохраняется. Реализовать через классы одного интерфейса, чтобы сначала реализовать на файлах, а потом мб переехать в бд.

После этого запустить процедуру переранжирования: сравниваю вакансию со всеми имеющимися резюме.


### Ранжирование 

Без ограничения общности далее рассмотрим на примере новой вакансии.

Алгоритм:
- беру краткие описания всех резюме
- составляю запросы к сетке. В идеале всё должно поместиться в один запрос (бейзлайн)
  - оценка нужно ли разбить на несколько запросов
    - кол-во использованных токенов
    - кол-во символов в запросов






### Список вакансий и подходящих под них резюме

Слева в меню нажимаю "список вакансий". 

Вижу список вакансий. Могу:
- провалиться в вакансию, чтобы посмотреть список подходящих кандидатов
- (минор) отфильтровать вакансии по департаменту, специальности, названию

Вижу список резюме, подходящих под вакансию. Могу:
- посмотреть предварительную оценку
- (минор) отфильтровать резюме по департаменту, специальности, названию
